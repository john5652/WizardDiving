# Wizard Diver Game - Design Document

## Game Concept

A 2D wizard exploration game inspired by Dave the Diver, where players fly through magical environments collecting spells, managing a magic school, and progressing through an AI-generated story.

## Core Gameplay Loop

1. **Exploration Phase:** Fly through magical environments (cities, castles, dungeons)
2. **Collection Phase:** Gather spells and resources
3. **Management Phase:** Return to magic school, manage classes, upgrade abilities
4. **Progression Phase:** Unlock new areas, learn new spells, advance story

## Key Features

### 1. Flying Exploration (Dave the Diver-style)

- **Movement:** Smooth flying mechanics through 2D environments
- **Environments:**
  - Magical cities (floating, vertical exploration)
  - Ancient castles (room-based, Castlevania-style)
  - Dark dungeons (puzzle-focused)
  - Mystical forests (open exploration)
- **Exploration Mechanics:**
  - Collect spells scattered throughout levels
  - Unlock doors using specific spells
  - Discover hidden areas
  - Find NPCs and quest givers

### 2. Spell System

- **Spell Collection:** Find spells while exploring
- **Spell Types:**
  - Combat spells (fireball, ice shard, lightning)
  - Utility spells (unlock door, reveal hidden, teleport)
  - Support spells (heal, shield, speed boost)
- **Spell Progression:**
  - Level up spells through use
  - Upgrade spells at magic school
  - Combine spells for new effects
- **Spell Inventory:** Limited slots, strategic choices

### 3. Magic School Management (Sushi Restaurant Replacement)

- **Class System:** Teach classes to earn resources
- **Resource Management:**
  - Mana crystals (currency)
  - Spell ingredients
  - Student satisfaction
- **Upgrades:**
  - Improve school facilities
  - Unlock new classes
  - Hire teaching assistants
  - Expand school building
- **Daily Cycle:** Similar to Dave the Diver's day/night cycle

### 4. Combat System

- **Enemy Encounters:** Monsters in exploration areas
- **Combat Mechanics:**
  - Real-time spell casting
  - Dodge and movement
  - Spell combinations
  - Weakness system (elemental)
- **Boss Battles:** Epic encounters requiring strategy

### 5. Quest & Story System

- **AI-Generated Content:**
  - Dynamic NPCs with unique personalities
  - Procedural quest generation
  - Branching storylines
- **Quest Types:**
  - Main story quests
  - Side quests from NPCs
  - Collection quests
  - Exploration quests
- **Dialogue System:** Multiple choice, relationship building

## Game Progression

### Early Game (Tutorial)
- Learn basic flying controls
- Collect first spells
- Understand school management basics
- Complete first quest

### Mid Game
- Unlock new areas
- Expand spell collection
- Upgrade school significantly
- Multiple quest lines active

### Late Game
- Master all spell types
- Fully upgraded school
- Epic boss encounters
- Complete main story
- Post-game content (new areas, challenges)

## Technical Design

### Art Style
- 2D pixel art or hand-drawn style
- Colorful, magical aesthetic
- Consistent visual language across biomes

### Controls
- **Keyboard:** WASD/Arrow keys for movement, Space for actions, Number keys for spells
- **Gamepad:** Full controller support
- **Mobile (future):** Touch controls with virtual joystick

### Performance Targets
- 60 FPS on target platforms
- Smooth scrolling and transitions
- Fast load times between areas

## Content Scope (MVP - 3 Months)

### Areas
- 3-5 distinct biomes
- 15-20 explorable rooms/areas
- 2-3 boss arenas

### Spells
- 20-30 unique spells
- 5-7 spell categories
- Upgrade paths for each

### NPCs & Quests
- 10-15 unique NPCs
- 20-30 quests (mix of main and side)
- AI-generated dialogue system

### School Management
- 5-7 class types
- 3-4 upgrade tiers
- Multiple school rooms to unlock

## Monetization Strategy (Future)

- **Steam/App Store:** Premium game ($9.99-$19.99)
- **DLC:** Additional biomes, spells, story content
- **Mobile:** Free with optional IAP (if adapted)

## Development Priorities

1. **Core Loop:** Flying, collecting, school management
2. **Polish:** Smooth controls, satisfying feedback
3. **Content:** Enough content for 5-10 hour playthrough
4. **Balance:** Fun progression, not grindy
5. **Story:** Engaging narrative with AI-generated variety
