## üìè How to Check Asset Sizes

### Method 1: Check the Asset Page (Before Downloading)

Most asset pages on OpenGameArt.org will tell you the size:

**Look for:**
- Title/description: "32x32 pixel art", "64x64 sprites"
- Image preview: Often shows dimensions
- File list: File names sometimes include size (e.g., `character_32x32.png`)
- Comments/discussion: Users often mention sizes

**Example from a typical page:**
- "This pack includes 32x32 pixel art characters"
- "Sprites are 64x64 pixels with transparent backgrounds"
- "Tiles are 16x16, 32x32, and 64x64 versions included"

### Method 2: Check After Downloading (Windows)

#### Using Windows File Explorer:

1. **Open the folder** containing the downloaded asset
2. **Right-click on the image file**
3. **Select "Properties"**
4. **Go to the "Details" tab**
5. **Look for:**
   - "Dimensions" (e.g., "32 x 32")
   - "Width" and "Height"

#### Using Windows Photos/Image Viewer:

1. **Right-click the image file**
2. **Select "Open with" ‚Üí "Photos"** (or your default image viewer)
3. **Right-click on the image** in the viewer
4. **Select "File info"** or look for properties
5. **Dimensions are usually shown**

#### Using File Explorer Preview Pane:

1. **Open File Explorer**
2. **Click "View" tab** ‚Üí Check "Preview pane" (right side)
3. **Click on an image file**
4. **Look at the preview pane** - it often shows dimensions

## üîç Quick Size Identification Tips

### What Sizes Are Common?

- **16x16 pixels** - Very small, retro-style
- **24x24 pixels** - Small sprites
- **32x32 pixels** - **Most common for 2D games** (your target)
- **48x48 pixels** - Medium size
- **64x64 pixels** - Larger, more detail
- **96x96+ pixels** - Very detailed

### What to Look For:

‚úÖ **Good signs:**
- Page says "32x32" or "32 pixel"
- Preview images look about right
- File names include "32" or "32x32"
- Comments mention the size

‚ùå **Don't worry if:**
- Size isn't mentioned (download anyway!)
- Size is slightly different (24x24, 48x48, 64x64)
- You can't tell the size (download and check later)
