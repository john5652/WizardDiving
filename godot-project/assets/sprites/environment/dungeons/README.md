# Dungeon Tilesets

## Files

- `dungeon_tileset_01.png` - Main dungeon/castle tileset
- `dungeon_tileset_02.png` - Modified/variant tileset

## Usage

See `TILESET_PROCESSING_COMPLETE.md` in the project root for detailed instructions.

## Quick Setup

1. Open tileset in Godot
2. Create TileSet resource
3. Define tile regions (32x32 or 16x16 recommended)
4. Use with TileMap nodes to build levels

## Elements Available

- Stone walls (various segments)
- Wooden platforms and stairs
- Doors, barrels, beds
- Windows and grates
- Waterfall and torch (animated)
- Character sprites
